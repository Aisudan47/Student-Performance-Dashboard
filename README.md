# 🎓 Student Performance Dashboard

 
<img src="student performance dashboard powerbi.png" alt="Alt text">

**Ever wondered what the story behind student performance numbers looks like?** This interactive Power BI dashboard transforms student data into clear, visual insights. Explore how test scores, mentorship involvement, academic choices, and student backgrounds connect. Whether you're aiming to improve education, analyze learning patterns, or build your data skills, this project provides a powerful lens.

---

## ✨ Our Goal: Making Sense of Student Data for Better Outcomes

This dashboard is built with a clear purpose:

* **Centralized Data View:** Think of it as your go-to spot for all key student performance and participation information, organized for easy access.
* **Identifying Key Trends:** We aim to highlight areas for improvement and the impact of mentorship programs by analyzing the data patterns.
* **Data-Driven Decisions:** Equip educators and administrators with evidence-based insights to make informed choices that support students.
* **Tracking Progress & Engagement:** Monitor important metrics like scores, participation time, and overall progress to foster a thriving learning environment.
* **Showcasing Data Skills:** This project demonstrates practical Power BI abilities applied to the meaningful field of educational data analysis.

---

## 🤔 Exploring Key Questions Through Data

This dashboard helps you find answers to crucial questions:

* **Performance by Group:** What are the average test scores when we look at different academic majors and genders?
* **Current Participation Levels:** How many students are currently enrolled and actively involved in learning activities?
* **Student Demographics:** Which countries are most represented within our student population?
* **The Impact of Mentoring:** Does having a mentor show a relationship with better student performance?
* **Engagement and Results:** Is there a connection between how much students engage and their academic achievements?

## 🚀 Getting Started with Data Exploration

Ready to explore the student data? Here's how:

1.  **Get the Project Files:**

    ```bash
    git clone [https://github.com/Aisudan47/Student-Performance-Dashboard.git](https://github.com/Aisudan47/Student-Performance-Dashboard.git)
    ```

2.  **Open the Dashboard File:**

    * Go to the `Dashboard/` folder and open `Student_Performance_Dashboard.pbix` using Power BI Desktop.

3.  **Connect to the Data (If Needed):**

    * If Power BI prompts you about the data source, navigate to `File` → `Options and settings` → `Data source settings`. Select the source and click `Edit` to point it to the `student_performance_data.xlsx` file on your computer.

4.  **Start Your Data Journey!**

    * Use the **filters** on the sides to focus your analysis.
    * **Hover** over charts to see more detailed data in the tooltips.
    * **Click** on charts (if they have the drill-down option) to explore the data in more detail.
